# Web Scraper Using Python

A simple Python web scraping project built for practicing Python programming skills and learning how websites and HTML parsing work.

## Project Goal

The purpose of this project is to improve my Python skills while learning concepts useful for future studies in AI and Machine Learning.

This project extracts useful information from websites by filtering unnecessary HTML content and saving the required data in a structured format.

---

## Libraries Used

### requests
Used to send HTTP requests and retrieve the HTML content of webpages.

### BeautifulSoup
Used to parse HTML pages and extract useful information in a clean and organized way.

---

## Features

- Fetch webpage HTML
- Parse webpage content
- Extract useful data
- Save extracted information
- Practice working with real-world data

---

## How It Works

1. Send a request to a website
2. Retrieve the HTML source code
3. Parse the HTML using BeautifulSoup
4. Extract required information
5. Store or display the extracted data

## Technologies Used

- Python
- requests
- BeautifulSoup4
- CSV / Pandas
